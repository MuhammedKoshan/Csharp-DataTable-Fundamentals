### 📝 Project: C# DataTable Fundamentals & Operations
---

This project is a comprehensive collection of practical examples demonstrating how to work with **ADO.NET DataTables** in C#. It showcases how to create, manipulate, and manage relational data entirely in-memory (disconnected architecture).

**Key Features & Examples Included:**
* **Creation & Setup:** Building offline DataTables from scratch.
* **CRUD Operations:** Adding, updating, clearing, and deleting rows programmatically.
* **Data Manipulation:** Filtering (searching) and Sorting data efficiently.
* **Aggregation Functions:** Performing calculations directly on data (Count, Sum, Avg, Min, Max).
* **Data Integrity:** Implementing constraints like Primary Keys and Auto-increment columns.

**Objective:** As a Computer Science student, this repository serves as a foundational building block for mastering ADO.NET in-memory data structures before transitioning to advanced ORM frameworks. It emphasizes clean code, structured learning, and solid backend data manipulation techniques.
